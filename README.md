
# PhoenixTask API

a simple project management system 


## Run Locally

Clone the project

```bash
  git clone https://github.com/ErfanMelon/PhoenixTaskApi.git
```

Go to the project directory

```bash
  cd PhoenixTaskApi
```

Build project

```bash
  dotnet build .\PhoenixTask.sln
```
Go To Web Folder
```bash
  cd .\src\WebApi\
```

Start the server

```bash
  dotnet run
```
